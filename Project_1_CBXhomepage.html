Home Page Re-Design

Redesign system home page with a better user experience approach

To provide the most appropriate contents and better user flow for users to improve their efficiency in daily work flow

My Role

  • User Research / UX Designer / UI Designer

Design Flow

Study Existing Design / Find out Problems

  1. Contents are not clear enough, confused about the meaning of differences between left panels and Tasks panel
  2. Task panel is hard to understand how to use it
  3. Not flexible enough to personalize the page

Problem Direction

 

Research Planning & Setup

Research Objectives

  1. To understand users' need for home page
    • What contents they might wanna see
    • What fields they need for each contents
    • What's the priority of the contents
  2. Know about the story of how users use home page for different contents

Questions

  1. What do users expect to do with Home Page and why?
  2. How do users use with Home Page?
  3. Pros for existing Home Page and why?
  4. Cons for existing Home Page and why?
  5. Prioritise the contents
  6. What do users think about the contents?
  7. What fields u wanna see in these contents?

 

User Research & Case Study

Target Users

Project Users

End users of the system. Using the system for their daily work. Including buyers and vendors.

Management Users

They are the team leads of the normal users. They need to keep track with the activities and work progress of team members.

Consultant

They are working with company's customers and users to gather requirements and help find solutions. They need to well familiar with the products and help the users to use the system.

Design Directions

Existing Feature improvements

  1. Business Activities
    • Use another UI presentation to replace bubbles
    • Combine them as one chart to save more space
    • Display numbers/counts for comparison
  2. Form
    • Use new forum style for consistency
  3. Alerts
    • Use a better layouts/style
  4. Critical Path Management (CRM)
    • Use another UI Presentation
    • Display by milestone tasks
    • Display exact counts
  5. Layouts
    • Data on a more outstanding place for eye-catching
    • Task/to-do will be placed together

New Features

  1. Shortcuts
    • Combine with My Favorite Listing View
    • Easy for users to access
    • Neat presentation
  2. Approvals
    • Easy for users to see how many approval documents he needs to do
    • See other approval documents
